# Madoka Launch Engine - Responsive Guidelines

## Design Strategy

**PC-First**: メインの作業はデスクトップ。スマホは限定的な機能のみ。

## Breakpoints

| 名称 | サイズ | 用途 |
|------|--------|------|
| sm | 640px | スマホ横向き |
| md | 768px | タブレット |
| lg | 1024px | ノートPC |
| xl | 1280px | デスクトップ（メインターゲット） |
| 2xl | 1536px | ワイドモニター |

## レイアウト構成

### Desktop (xl以上)

```
┌──────┬──────────────────────────────────┐
│      │                                  │
│ Side │       Main Content               │
│ bar  │       (max-width: 1200px)        │
│      │                                  │
│256px │                                  │
│      │                                  │
└──────┴──────────────────────────────────┘
```

### Laptop (lg)

```
┌────┬────────────────────────────────────┐
│    │                                    │
│Side│       Main Content                 │
│64px│       (fluid)                      │
│    │                                    │
└────┴────────────────────────────────────┘
```
- サイドバー折りたたみ（アイコンのみ）

### Tablet (md)

```
┌────────────────────────────────────────┐
│  Top Navigation Bar                     │
├────────────────────────────────────────┤
│                                        │
│       Main Content (fluid)             │
│                                        │
└────────────────────────────────────────┘
```
- サイドバー → ハンバーガーメニュー + オーバーレイ

### Mobile (sm以下)

```
┌────────────────────┐
│  Top Bar + Menu     │
├────────────────────┤
│                    │
│  Main Content      │
│  (single column)   │
│                    │
├────────────────────┤
│  Bottom Navigation │
└────────────────────┘
```

| 数値入力 | Full | Full | **Full** | P0 |
| 配信文プレビュー | 2col | 1col | **1col** | P0 |

## Grid System

### プロジェクトカード

```css
/* Desktop */
grid-template-columns: repeat(3, 1fr);
gap: 24px;

/* Tablet */
@media (max-width: 1024px) {
  grid-template-columns: repeat(2, 1fr);
  gap: 16px;
}

/* Mobile */
@media (max-width: 768px) {
  grid-template-columns: 1fr;
  gap: 12px;
}
```

### KPIカード

```css
/* Desktop */
grid-template-columns: repeat(3, 1fr);
gap: 24px;

/* Mobile */
@media (max-width: 768px) {
  grid-template-columns: 1fr;
  gap: 12px;
}
```

### 2カラムエディタ（LP編集）

```css
/* Desktop */
grid-template-columns: 280px 1fr;
gap: 0;

/* Desktop以下 */
/* モバイル非対応 - デスクトップのみ */
```

## タッチ対応

| 要素 | 最小タップターゲット | 間隔 |
|------|---------------------|------|
| ボタン | 44px | 8px |
| リスト項目 | 48px | 0 (dividerで分離) |
| チェックボックス | 44px | 8px |
| テキスト入力 | 44px高 | - |
| タブ | 44px高 | 0 |

## パフォーマンス

| 対策 | 説明 |
|------|------|
| 画像最適化 | Next.js Image、WebP、lazy loading |
| コンポーネント分割 | 動的import（ファネルビジュアライザー、チャート等） |
| 仮想化 | 長いリスト（配信一覧、ナレッジ）は仮想スクロール |
| フォント | Noto Sans JPのサブセット化（日本語フォント最適化） |
| prefetch | リンク先のprefetch（ページ遷移高速化） |
